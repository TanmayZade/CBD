## Overview
The CBD Project is a Python-based application designed to utilize machine learning models, specifically the cb_sgd_final.sav file, for predictive analysis. This project aims to provide an efficient and user-friendly interface for users to interact with the models, with the help of the app.py file. The project structure is well-organized, making it easy for new users to navigate and contribute.
